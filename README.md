# Car vs Motorcycle Image Classification

## Overview

This project is an AI image classification model developed using Google Teachable Machine and TensorFlow. The model is trained to classify images into two categories:

- Car
- Motorcycle

The project was developed and tested using Google Colab.

---

## Technologies Used

- Python
- TensorFlow / Keras
- Google Teachable Machine
- NumPy
- Pillow
- Google Colab

---

## Project Files

- `Car_vs_Motorcycle_Classification.ipynb` – Google Colab notebook.
- `keras_model.h5` – Trained AI model.
- `labels.txt` – Class labels.
- `images.jpeg` – Sample image used for testing.

---

## How to Run

1. Open the notebook in Google Colab.
2. Upload the required files (`keras_model.h5`, `labels.txt`, and `images.jpeg`).
3. Run all notebook cells.
4. The model will predict whether the image is a **Car** or a **Motorcycle**.
